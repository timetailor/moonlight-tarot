달빛타로 데이터 삭제 안내

달빛타로는 사용자의 타로 질문, 리딩 기록, 오늘의 카드 기록 및 앱 설정값을 별도의 자체 서버에 저장하지 않으며, 해당 데이터는 사용자 기기 내부에 저장됩니다.

앱 내 데이터 삭제 방법

사용자는 달빛타로 앱의 리딩 기록 화면에서 개별 리딩 기록을 삭제할 수 있습니다.

또한 환경 설정 화면의 ‘전체 리딩 기록 삭제’ 기능을 통해 저장된 리딩 기록을 한 번에 삭제할 수 있습니다.

앱 삭제를 통한 데이터 삭제

사용자가 기기에서 달빛타로 앱을 삭제하면, 기기 내부에 저장된 리딩 기록, 오늘의 카드 기록 및 앱 설정 데이터가 함께 삭제됩니다.

삭제 또는 보관되는 데이터

  • 삭제 대상: 타로 리딩 기록, 오늘의 카드 기록, 앱 설정값
  • 보관 데이터: 달빛타로는 자체 서버에 사용자 데이터를 별도로 보관하지 않습니다.
  • 추가 보관 기간: 자체 서버 보관 데이터가 없으므로 별도의 추가 보관 기간은 없습니다.
